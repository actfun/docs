ACTFUN runs on Arc Testnet, a test network where all transactions use USDC — a testnet currency with no real monetary value. Before you can mine or trade, you need MetaMask installed, Arc Testnet added to your wallet, and some free USDC from the faucet. This page walks you through each step.

<Note>
  Arc Testnet is a test environment. Testnet USDC tokens have no real monetary value and exist solely for testing. Never send real funds to testnet addresses.
</Note>

## Prerequisites

- MetaMask installed in your browser ([metamask.io](https://metamask.io))
- A few minutes to complete the setup below

---

## Setup

<Steps>
  <Step title="Add Arc Testnet to MetaMask">
    MetaMask does not include Arc Testnet by default. Add it manually using the network details below.

    1. Open MetaMask and click the network selector at the top of the extension.
    2. Click **Add network**, then **Add a network manually**.
    3. Enter the following values and click **Save**.

    | Field | Value |
    | --- | --- |
    | Network name | Arc Testnet |
    | RPC URL | `https://rpc.testnet.arc.network` |
    | Chain ID | `5042002` |
    | Currency symbol | `USDC` |
    | Block explorer URL | `https://testnet.arcscan.app` |

    After saving, switch to **Arc Testnet** in the network selector. Your balance will show `0 USDC` until you claim from the faucet.

    <Tip>
      If MetaMask shows a "Chain ID already exists" error, you already have Arc Testnet added. Switch to it from the network list.
    </Tip>
  </Step>
  <Step title="Get free USDC from the faucet">
    You need USDC to pay mining fees. Claim free testnet USDC from the official faucet.

    1. Go to [faucet.testnet.arc.network](https://faucet.testnet.arc.network).
    2. Paste your MetaMask wallet address into the input field.
    3. Complete any verification step shown and click **Request**.
    4. Wait a few seconds, then check your MetaMask balance — you should see USDC arrive.

    <Note>
      The faucet has a per-address rate limit. If you run out of USDC during testing, return to the faucet and request again after the cooldown period.
    </Note>
  </Step>
  <Step title="Connect your wallet on ACTFUN">
    1. Open [actfun.fun](https://actfun.fun) in your browser.
    2. Click **Connect wallet** in the top-right corner.
    3. MetaMask will prompt you to select an account. Choose your account and click **Connect**.
    4. If MetaMask asks you to switch to Arc Testnet, approve the network switch.

    Once connected, your address and USDC balance appear in the header.

    <Warning>
      If the app shows a "Wrong network" banner, open MetaMask and switch to **Arc Testnet** manually. ACTFUN only works on Chain ID 5042002.
    </Warning>
  </Step>
  <Step title="Browse tokens and mine one">
    The homepage shows all tokens currently available to mine. Each card displays the token name, symbol, mining progress, and USDC fee per mine.

    1. Click any token card to open its detail page.
    2. Read the token description and check the mining progress bar. Tokens below 95% are still mineable.
    3. Type a funny post in the **Mine** panel — this is your on-chain message recorded with the transaction.
    4. Click **Mine**. MetaMask will ask you to confirm a transaction that sends the USDC fee to the contract.
    5. Approve the transaction. Once confirmed, the mined tokens appear in your wallet.

    <Tip>
      Each token has a per-wallet cooldown set by its creator. If the **Mine** button is disabled after your first mine, wait for the cooldown to expire before mining again.
    </Tip>
  </Step>
</Steps>
