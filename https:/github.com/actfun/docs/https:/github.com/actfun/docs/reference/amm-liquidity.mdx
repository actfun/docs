When a token graduates, ACTFUN seeds liquidity on three AMMs simultaneously — every protocol is source-verified on Arc Explorer.

## UnitFlow V3 — Concentrated Liquidity

The primary AMM. A Uniswap V3 fork with full-range positions and sub-second finality on Arc. Every graduated token gets a dedicated V3 pool seeded at launch.

**Fee tier:** 0.30% · **Ticks:** `-887220` / `887220` (full range)

## Uniswap V2 — Classic Constant-Product

The battle-tested `x * y = k` AMM. A second pool is created per token at graduation, giving traders a familiar swap interface and deep compatibility with V2-based tooling.

## Curve Finance (StableSwap) — Low-Slippage Pools

A Curve-style StableSwap pool is deployed per token via the factory at graduation. Each token gets its own dedicated pool — designed for low-slippage swaps on correlated or newly-bootstrapped pairs.
